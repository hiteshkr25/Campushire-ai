CSV File Layout Guidelines

The parser expects the CSV file to contain a header row. Columns change depending on the roles:

Role Column Required Columns / Field Parameters
student `email`, `role` (student), `password`, `first_name`, `last_name`, `phone`, `college_code` (e.g. IITB), `branch_code` (e.g. CSE), `enrollment_number`, `batch` (e.g. 2023-2027), `graduation_year`
recruiter `email`, `role` (recruiter), `password`, `first_name`, `last_name`, `phone`, `company_name`, `designation`
tpo `email`, `role` (tpo), `password`, `first_name`, `last_name`, `phone`, `college_code` (e.g. IITB), `designation`, `department`
admin `email`, `role` (admin), `password`
