# System Features Catalog

CampusHire AI is a Training & Placement platform integrated with AI-driven resume parsing and recruitment analytics.

---

## Key Features Directory

### 1. Student Portal Module
- **Profile Management**: Build personal portfolios, project registries, and certification records.
- **Academic Profile Lock**: Verification by TPO locks academic details to prevent edits.
- **Resume Versioning**: Maintain multiple resumes and select one active copy for applications.
- **AI Resume Parser**: Automatic extraction of skills, education, and experience from PDF uploads.
- **ATS Rank Matcher**: Displays real-time ATS match scores and suggests improvements.

### 2. Recruiter Portal Module
- **ATS Candidate Directory**: Search and filter applicants by cgpa, backlogs, skills, or drive.
- **Candidate Comparison**: Compare up to 3 candidates side-by-side with match scores and skill gaps.
- **Workflow Scheduler**: Manage multi-round hiring schedules and record feedback.
- **Offer Engine**: Release job offers, upload salary letters, and track responses.
- **Recruitment Funnel**: Visual analytics tracking candidates through the hiring pipeline.

### 3. TPO Portal Module
- **Verification Engine**: Review student academic records and approve profiles.
- **Partners Directory**: Manage corporate partner companies and logos.
- **Campaign Manager**: Design recruitment drives with targeted eligibility requirements.
- **Eligibility Engine**: Automatic matching of eligible students for active campaigns.
- **Placement Analytics**: Interactive reports on branch hires, average salaries, and company analytics.

### 4. Admin Portal Module
- **System Dashboard**: Track platform status, user registrations, logins, and storage.
- **User Directory**: Search, edit roles, reset passwords, and lock/unlock accounts.
- **Bulk Imports**: Upload student and recruiter rosters via CSV.
- **Audit Logging**: Secure logs tracking user actions and security events.
